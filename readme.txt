== Description ==

This plugin provides a blank template, for pages and posts.
The template is totally empty, which means you will escape any heavy HTML, CSS, JS loading, if you're using a visual editor or a heavy template.

== Frequently Asked Questions ==

= Do you load any HTML/CSS/JS in the template =

No, it is clean as it can get.
